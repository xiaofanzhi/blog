# fzx-blog
基于Python Flask框架的人博客



**`Flask 扩展插件`**

- Flask-SQLAlchemy 数据库模型
- Flask-Admin 后台管理




**`虚拟环境`**

开启 pipenv shell

**`数据库`**

命令：export FLASK_APP=test.py （选定当前运行为test.py）

初始化:flask db init

迁移:flask db migrate

升级:flask db upgrade




**文章相关**

- 文章列表展示
- Flask-Admin 修改文章集成[editor.md](https://github.com/pandao/editor.md) Markdown编辑器
